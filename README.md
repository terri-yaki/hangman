# Hangman

A simple implementation of the classic word guess game Hangman

## Description

This repository contains a Python-based version of Hangman, a popular word guessing game where players try to guess the letters of a hidden word one letter at a time. The game continues until the player can guess the word or a hangman is drawn.

## Getting Started

### Dependencies

* Python 3.x

### Installing

* Clone this repository to your local machine.
* Make sure Python is installed on your system.

### Executing program

* Open your terminal or command prompt.
* Navigate to the directory where you have cloned this repository.
* Run the following command:
```bash
python hangman.py
```
